# Attendance-system
A smart class attendance system using Bluetooth-based attendance and dashboards for students and lecturers.


1. **Clone the repository (first time only):**  
```bash
git clone https://github.com/Techhigh-org/Attendance-system.git
cd Attendance-system
```

2. **Pull the latest changes before working:**  
```bash
git pull origin main
```

3. **Create and switch to their task branch (example: `student-dashboard-ui`):**  
```bash
git checkout -b student-dashboard-ui
```

4. **Add code changes locally.**

5. **Stage all changes:**  
```bash
git add .

6. **Commit with a clear message:**  
bash
git commit -m "Added student dashboard UI"
7. **Push their branch to GitHub:**  
```bash
git push origin student-dashboard-ui
